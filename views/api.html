React Board - API 목록

인증

  • 로그인: [POST] /api/auth/login
  • 회원가입: [POST] /api/auth/register

유저

  • 유저 목록 (개발용): [GET] /api/users
  • 유저 상세: [GET] /api/users/:userId

게시판

  • 게시글 목록: [GET] /api/posts
  • 내 게시글 목록: [GET] /api/posts/my
  • 게시글 상세: [GET] /api/posts/:postId
  • 게시글 작성: [POST] /api/posts
  • 게시글 수정: [PUT] /api/posts
  • 게시글 삭제: [DELETE] /api/posts

댓글

  • 내 댓글 목록: [GET] /api/comments/my
  • 게시글 댓글 목록: [GET] /api/comments/posts/:postId
  • 댓글 작성: [POST] /api/comments/posts/:postId
  • 댓글 수정: [PUT] /api/comments/:id
  • 댓글 삭제: [DELETE] /api/comments/:id
